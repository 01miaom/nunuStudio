Particles

nunuStudio particle engine is powered by the SPE engine, the particle object encapsulates a SPE.Emitter and a SPE.Group that can be easily edited in the Particle Editor, but more Emitters can be added progamatically during runtime.

Documentation about the SPE engine can be found here.

To add a new particle object select it from the tool bar. All new particles are created with the particle fountain template.

Particle Editor

The particle editor can be used to customize every aspect of a particle object using a visual tool.

To open the particle editor, simply doubli click on the particle object in the object explorer
