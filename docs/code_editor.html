Code Editor

The nunu studio code editor is build on top of codemirror (available at codemirror.net), it includes syntax highligthing, code linter, search and replace, code identation (with auto code identation), inteligent code completion (based on previous written code), and shortcut support.

Shortcuts

- CTRL + F Find words or regex
- CTRL + SHIFT + F Search and replace
- CTRL + SHIFT + F Search and replace all
